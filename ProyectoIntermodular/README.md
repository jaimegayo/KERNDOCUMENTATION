# Kern App - Frontend (Android)

Aplicación móvil desarrollada en **Java** para la gestión de rutinas de entrenamiento.

## 📱 Características
- Login y Registro integrados con API REST.
- Cuestionario inicial de salud y objetivos.
- Navegación basada en Fragments y Navigation Component.

## 🛠️ Configuración
- Abrir el proyecto en **Android Studio**.
- Asegurarse de que el `BASE_URL` en `UserRepository.java` apunta a la IP correcta del servidor o a la URL de Vercel.
- SDK mínimo: 24 (Android 7.0).

## 📚 Bibliotecas principales
- **Retrofit 2**: Para las peticiones HTTP.
- **Gson**: Para el parseo de JSON.
- **Navigation Component**: Para el flujo entre pantallas.
